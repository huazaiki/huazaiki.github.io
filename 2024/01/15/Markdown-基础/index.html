    写在开头

    这是我的博客重置之后的第一篇文章，在新的一年里我想对自己学习知识的过程进行整合，整合需要一个媒介（工具），所以我选择学习markdown来编写md文档。

    Markdown历史简述

    Markdown由John Gruber在2004年发布
    Markdown是一种语法标记语言，本质上是对HTML的一种化简
    Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档

    Markdown基础语法

    标题语法

    1. 以 # 开头，后接内容
    2. 有1-6级标题，级别对应其 # 的个数
      # 一级标题                      <h1>一级标题</h1>
## 二级标题         对应HTML        ...
### 三级标题        =======> 
#### 四级标题
##### 五级标题
###### 六级标题
    3. # 与标题内容间须留有一个空格
    4. Setext格式标题（另一种格式的标题）
      （1）注明一下，上述用 # 表示标题的为 ATX样式
      （2）文字下面加任意多个 = 表示一级标题
      （3）文字下面加任意多个 - 表示二级标题

    段落语法

    1. 文本后加两个空格表示换行（实测VScode配合MPE无效）
    2. 多个连续空格只会被解析为一个空格
    3. 保留多个空格使用HTML语法中的空格字符
      • &nbsp
      • &emsp
    4. 表示两行需要在两行之间加一空行
    5. 引言使用 >

      这是引言

      asd
      结束引用块加一空行

    无序列表

    1. 使用 ***、+、-**表示列表项
    2. 使用缩进实现子列表项
    3. 项中要包含内容需要换行然后缩进至项下

    有序列表

    1. 数字加点 后接空格 再接内容（也可以数字加 ) 后接空格 再接内容）
    2. 有序列表项数字顺序至于第一个数字有关
      1. ...      1. ...          1. ...
2. ...  or  3. ...  ===>    2. ...
3. ...      3. ...          3. ...
    3. 有序列表和无序列表可以互相嵌套

    分割线

    1. 使用 *、-、_ 中任意一个字符至少重复三次
    2. 对应HTML中 <hr/>
    3. 分割线上下建议加上空行
    4. 特别说明 - 分割线上方不要有内容（Setext标题）

    代码块

    1. 仅建议使用三个反引号（通用键盘上数字1左边的按键）或三个波浪线
    2. 反引号后面可以添加代码语言，可以被渲染高亮
      print("Hello, world")

    行内标记

    1. *内容* ==> 内容
    2. **内容** ==> 内容
    3. ***内容*** ==> 内容
    4. `内容` ==> 内容
    5. ~~内容~~ ==> 内容
    6. <u>内容</u> ==> 内容

    图片插入

    1. 格式为：![描述](图片位置)
    2. 位置可以是链接也可以是本地图片路径
    3. markdown文件不会存储图片，只作为标记（如果想要转发md文件记得打包转发）
    4. 想要调整大小样式，使用HTML的<img>标签

    链接插入

    1. 格式为：[描述](链接)
    2. 对应HTML中的 <a href:”链接”>描述</a>
