    5

    40    
    >>> largest_factor(13) # factor is 1 since 13 is prime  
    1    
    """

        factor = 2  
    while factor ** 2 <= n:  
        if n % factor:  
            factor += 1  
        else:  
            return int(n / factor)

    标准答案：

        factor = n - 1
    while factor > 0:
        if n % factor == 0:
            return factor
        factor -= 1

    Note : 要检查 a 是否被 b 除尽，可以使用表达式 a % b == 0，它可以读作“a 除以 b 的余数是 0
    Note_2 : 因子总是成对出现，所以循环终止条件是 factor ** 2 > n, 并且在返回值的时候是返回 n / factor(例如15 / 3 = 5)，确保返回值是整数使用 int()包装
