    1. 基础命令

    此笔记用于未来温习Git相关指令及操作

    基础命令

    1. 初始化git仓库
    git init
    1. 添加至暂存区
    git add 'file_name'
#或使用下面指令添加当前文件夹所有文件 
git add .
    1. 提交至本地仓库
    git commit -m 'your comment'

#对已经添加过的文件可以通过以下命令直接提交
git commit -a -m 'your comment'
    1. 查看当前工作目录 git工作状态
    git status
    1. 显示日志信息
    git log

#单行显示
git log --oneline
#以图的形式显示
git log --graph
    1. 展示特定提交信息
    git show 'git-commit-id'
    1. 跳转到特定提交
    git reset --hard 'git-commit-id'
    1. 查看本地日志（关闭终端还有的）
    git reflog
    1. 创建分支
    git branch

# 对应的删除分支是
git branch -d 'branch name'
    1. 合并分支
    git merge 'branch name'
    1. 查看分支合并冲突
    git diff
    1. 变基分支
    git rebase master
    1. 优选
    git cherry-pick <commit id>:单独合并一个提交
    1. 克隆项目
    git clone 远程仓库地址
    1. 抓取、拉取和冲突解决
    git fetch 远程仓库 #抓取：只获取但不合并远端分支，后面需要我们手动合并才能提交
git pull 远程仓库 #拉取：获取+合并
